# v2 mockup — 컨베이어 전 스냅샷

`feat/frontend-prototype` 브랜치의 커밋 `e48bff7`(2단계 컨베이어 라운드 전환 도입 **직전**) 시점

  이 폴더에서 재빌드하려면 경로를 v1로 맞춰야 함. 보관·참조용.
